# 项目简介

## 核心功能
- 音乐播放量排行榜
- 播放歌曲
- 查看音乐信息
- 用户注册登录
- 管理员管理用户

## 适用场景
- 音乐爱好者
- 音乐网站后台管理
- 音乐社区建设

## 技术特点
- 采用Python编程语言，Django框架，MySQL数据库
- 具备良好的代码结构和可维护性
- 提供详细的部署教程，易于上手运行
